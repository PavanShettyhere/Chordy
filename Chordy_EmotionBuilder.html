Integration Guide
Step 1 — Add to AnimID enum in Config.h:
ANIM_<YOUR_NAME>, // before ANIM_COUNT

Step 2 — In animPlay(), add a case:
case ANIM_<NAME>: animFrameDelay=...; animLoop=...; break;

Step 3 — In animTick(), add your case block.
The generated code includes a ready-to-paste block.

Step 4 — To trigger from web, add a button to the
trig-grid div in WebDashboard.h with the new ID.

Step 5 — Add to idle pool in stateMachineTick()
if you want it to appear randomly.
