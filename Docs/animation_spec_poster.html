State Machine — Reactive Logic
STATE ENTER CONDITION EXIT CONDITION ANIMATION PLAYED SOUND
IDLE Default / after 8s from any state Any trigger fires IDLE, BLINK, LOOK_L/R (random) None
HAPPY WiFi connected, timer done 8s timeout → IDLE HAPPY (hands + bounce) Happy melody
VERY HAPPY BTN_INTERACT, 2h PIR absence 8s timeout → IDLE VERY_HAPPY (hearts) Happy melody (high)
SCARED LDR delta < –600 (sudden dark) 60 frames → IDLE SCARED (shake + sweat) Scared sound
BRIGHT LDR delta > +600 (sudden bright) Squint anim ends → IDLE SQUINT Short pip
SLEEPY BTN_POWER press BTN_POWER again SLEEPY (zzz loops) Sleep melody
WEATHER_COLD Fetched temp ≤ 15°C 80 frames → IDLE WEATHER_COLD (blanket) None
WEATHER_HOT Fetched temp ≥ 28°C 80 frames → IDLE WEATHER_HOT (fan) None
WEATHER_RAIN OWM code 500–599 80 frames → IDLE WEATHER_RAIN (drops) None
TIMER_RUNNING BTN_EXTRA or web timer Countdown hits zero IDLE (timer bar on web) Timer alarm when done
